# GerenciadorSimples
Este repositório contém o projeto GerenciadorSimples, o qual é um sistema voltado para a gestão de tarefas desenvolvido com foco na integração entre uma API Rest em Spring Boot, e uma interface SPA utilizando o framework VueJs.
Grupo: Nycolas Rozisca Moreno e João Rafael.

Divisão de tarefas:
Nycolas: Lógica de backend, o que inclui a criação das rotas da API, a implementação do CRUD, e a configuração da persistência no banco de dados.
João: Frontend em VueJs, lógica de navegação e criação das interfaces do sistema.

As três telas que compõem a interface do usuário, e que devem ser desenvolvidas pelo responsável do frontend são a tela de login, para a simulação de acesso do usuário ao sistema, a tela de dashboard, que apresentará um resumo visual e as informações gerais das atividades, e a tela de gerenciamento, onde o usuário poderá realizar as operações de criar, listar, atualizar, e excluir as tarefas, integrando-se diretamente com o backend para os dados serem salvos no banco.
